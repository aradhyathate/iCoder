# 🚀 iCoder - Heaven for Programmers

A modern and responsive multi-page website built using **HTML5**, **CSS3**, and **Bootstrap 5**.
Designed to provide a clean user experience and demonstrate front-end development skills through a professional and visually appealing interface.

---

## 📌 Overview

**iCoder** is a responsive website that showcases how Bootstrap components and layouts can be used to create modern web pages efficiently. The project includes multiple pages with an intuitive navigation structure and a user-friendly design.

---

## ✨ Features

* ✅ Fully Responsive Design
* ✅ Modern UI using Bootstrap 5
* ✅ Multi-page Website Structure
* ✅ Interactive Navigation Bar
* ✅ About Section
* ✅ Contact Page
* ✅ High-Quality Images and Layout
* ✅ Clean and Readable Code Structure
